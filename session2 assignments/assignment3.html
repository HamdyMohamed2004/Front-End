!important Attribute in CSS

The !important keyword in CSS is used to increase the priority of a specific style rule. When appended to a CSS property value, it ensures that this rule will override any other rules that target the same element, regardless of their specificity or source order.

Example : 

.bg-primary {
  background-color: green !important;
}

This code in css make the Background color is green, although bg-primary in bootstrap make Background color is Blue
